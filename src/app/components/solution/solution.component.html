The Solution

Imagine a snap on helmet mirror that addresses all of the limitations existing bike mirrors have, such as: Vibration; A multitude of adjustments to suit all riders; An easy snap on/ snap off format; A novel marking system which allows for quick reconfiguring if required on a new ride; AND; The ability to see from any position on the bike a rider might have. e.g., from standing to more aero dynamic positions – all without having to readjust the previous set mirror system!
